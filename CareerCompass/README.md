# CareerCompass - بوصلة المسار

## نظام التوجيه الأكاديمي للطلاب الأردنيين

نظام توجيه أكاديمي متطور يساعد الطلاب الأردنيين (الصفوف 9-12) على اختيار المسار التعليمي والتخصص الجامعي الأنسب بناءً على تقييم علمي ثلاثي الأبعاد.

## 🎯 المميزات الرئيسية

- ✅ نموذج تقييم ثلاثي الأبعاد (الميول + القدرات + أسلوب العمل)
- ✅ 48 سؤالاً علمياً دقيقاً
- ✅ توصيات مخصصة حسب الصف الدراسي
- ✅ خطة تطوير شخصية للصفوف 9-11
- ✅ توصيات مباشرة بالتخصصات الجامعية للصف 12
- ✅ دليل شامل للجامعات الأردنية
- ✅ واجهة عربية كاملة RTL
- ✅ تصميم متجاوب (Desktop & Mobile)

## 🛠 التقنيات المستخدمة

- **Vue.js 3** - مع Composition API
- **Vite** - أداة البناء السريعة
- **Tailwind CSS** - للتصميم
- **Pinia** - إدارة الحالة
- **Vue Router** - التوجيه
- **Google Fonts** - Cairo & Tajawal (خطوط عربية)

## 📦 التثبيت والتشغيل

- Node.js (v16 أو أحدث)

1. **استنساخ أو تنزيل المشروع**

## 🧭 آلية العمل

### النموذج الثلاثي الأبعاد

#### 1️⃣ الميول والاهتمامات (20 سؤال)
- Realistic: العمل العملي والتطبيقي
- Investigative: البحث والتحليل العلمي
- Social: مساعدة الآخرين والتفاعل
- Conventional: العمل المنظم والدقيق
- Enterprising: القيادة والمبادرة
- Artistic: الإبداع والتعبير الفني

#### 2️⃣ القدرات والمهارات (16 سؤال)
- Math: المهارات الرياضية
- Biology: العلوم الحيوية
- Precision: الدقة والانتباه للتفاصيل
- Communication: التواصل
- Technology: المهارات التقنية
- Stress Tolerance: التعامل مع الضغط
- Analytical: التفكير التحليلي
- Organization: التنظيم

#### 3️⃣ أسلوب العمل (12 سؤال)
- Team vs Solo
- Routine vs Dynamic
- Quiet vs Active
- Indoor vs Field
- Decision Making vs Instructions

### معادلات التوصيات

```javascript
// مثال: المجال الصحي
Health Score = (Social × 0.3) + (Biology × 0.25) + (Precision × 0.2) + 
               (Stress × 0.15) + (Communication × 0.1) + 
               (Grades Bonus × 0.2)

// مثال: الهندسة
Engineering Score = (Realistic × 0.3) + (Math × 0.3) + 
                    (Technology × 0.2) + (Analytical × 0.2)
```

## 📝 الملاحظات الفنية

### RTL Support
- جميع المكونات تدعم RTL
- `dir="rtl"` مطبق عالمياً
- خطوط عربية (Cairo & Tajawal)

### Responsive Design
- تصميم متجاوب بالكامل
- نقاط توقف: Mobile, Tablet, Desktop
- قوائم تنقل مناسبة للجوال

### State Management
- Pinia stores: `authStore` و `assessmentStore`
- حفظ تلقائي في localStorage
- استرجاع البيانات عند إعادة التحميل

### Router Guards
- حماية المسارات التي تحتاج مصادقة
- إعادة توجيه تلقائية للصفحة الرئيسية
